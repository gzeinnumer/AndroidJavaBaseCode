# Download

Add maven `jitpack.io` and `dependencies` in `build.gradle (Project)` :

```gradle
// build.gradle project
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}

// build.gradle app/module
dependencies {
  ...
  implementation 'com.github.gzeinnumer:AndroidJavaBaseCode:version'
}
```
